SocFlow 계정을 만들고 첫 게시물을 올려보세요.

### SocFlow 란?

SocFlow는 "소셜 계정을 한곳에서 관리"를 목표로 만들어진 플랫폼입니다.
이 글에서는 SocFlow 계정을 만들고, 첫 게시물을 작성해 공개하기까지의 과정을 차근차근 안내합니다.

---

## 1. SocFlow 계정 만들기

SocFlow를 사용하려면 먼저 계정을 생성해야 합니다.

1. SocFlow 홈페이지에 접속합니다. -> [SocFlow 공식 사이트](https://socflow.app)

<Image src={oneImage} alt="SocFlow 공식 사이트" />

2. 오른쪽 상단에 있는 시작하기 버튼을 클릭합니다.

<Image src={twoImage} alt="SocFlow 회원가입 화면" />

3. 이메일 / 소셜 로그인 중 하나를 선택합니다

4. 계정 약관 동의하여 생성을 완료합니다.

- 회원가입 완료하면 계정 약관 동의하는 페이지로 이동됩니다.

---

## 2. 대시보드 살펴보기

로그인이 완료되면 대시보드로 이동합니다.
대시보드에서는 다음과 같은 기능을 확인할 수 있습니다.

- 게시물 관리
- 소셜 계정 관리
- 미디어 관리
- API Key 관리
- 설정

<Image src={threeImage} alt="SocFlow 대시보드 화면" />

처음이라면, 전체 구조를 한 번 가볍게 구경하는 것을 추천합니다.

---

## 3. 첫 게시물 작성하기

이제 첫 게시물을 작성해봅시다.
혹시 연동한 소셜 네트워크 계정이 없다면 다음 글을 참고하세요.
[SocFlow 소셜 네트워크 계정 연동하기](https://blog.socflow.app/blog/social-connect)

1. 게시물 관리 -> 게시물 작성 페이지로 이동합니다.

<Image src={fourImage} alt="SocFlow 계정 선택 화면" />

2. 업로드할 계정을 선택해줍니다.

3. 포스팅 타입을 선택하고 내용을 작성합니다.

<Image src={fiveImage} alt="SocFlow 내용 작성" />

해시태그 등도 작성하면 효과가 좋습니다.
미디어도 당연히 추가할 수 있고요.

4. 즉시 발행 클릭

## 모든게 너무 쉽고 완벽합니다.
